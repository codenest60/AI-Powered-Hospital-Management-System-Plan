# 🏥 AI-Powered Hospital Management System

## 📖 Overview

This project is an AI-Powered Hospital Management System designed to streamline hospital operations including patient registration, doctor scheduling, appointment management, medical records, billing, and AI-assisted symptom checking. It helps hospital staff deliver better healthcare efficiently.

---

## 🎯 Features

* Patient Management
* Doctor Management
* Appointment Management
* Medical Records Management
* Billing & Invoice Management
* AI Symptom Checker
* Prescription Management
* Report Generation

---

## 👥 Users

### Admin

* Manage patients and doctors
* Manage departments
* Generate hospital reports
* Publish announcements

### Doctor

* View and manage appointments
* Write prescriptions
* Update patient medical records
* View patient history

### Patient

* Register and manage profile
* Book appointments
* View medical records
* Use AI symptom checker
* View billing & invoices

### Receptionist

* Register new patients
* Schedule appointments
* Manage billing

---

## 🛠️ Technology Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB
* AI: OpenAI API
